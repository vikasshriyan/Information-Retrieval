The program requires an external library called JSOUP (jsoup-1.8.3.jar), which is provided with my code. Remember to keep both the files while running the program.

Compile the code using the following command:
javac -cp jsoup-1.8.3.jar WebCrawler.java

Execute the code using the following command:
java -cp .:jsoup-1.8.3.jar WebCrawler http://en.wikipedia.org/wiki/Hugh_of_Saint-Cher

After completion of the program, it will output the Unique URLs found
and the number of the URLs found.
